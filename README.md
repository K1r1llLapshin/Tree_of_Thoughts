# Дерево мыслей (Tree of Thoughts)

### Общая информация:
    - Для работы с LLM используется API от AITUNNEL: https://aitunnel.ru/
    - Модель: deepseek-chat-v3
    - Для визуалиции дерева мыслей используеся graphviz: https://graphviz.org/
    
### Генерация и оценка мысли:
    1. Для генерации мысли используются 3 промта:
        - START_PROMPT: начальный промт, которы разбтвает исходную задачу на k первых мыслей
        - THOUGHT_PROMPT: промт, который генерирует следующие мысли
        - SUMMARY_PROMPT: промт, который подводит результаты генерации мысли
    2. Для оценки мысли используются промт EVALUATOR_PROMPT, который оценисает по шкале от 0.0 до 1.0

### Для поиска лучших мысле реализованы следующие обходы:
1. BFS  
    - Параметры: 
        - count_thoughts: количество генерируемых мыслей на каждом шаге
        - breadth_limit: количество лучших мыслей оставляем для следующего уровня
        - max_depth: максимальная глубина дерева
    - На каждом шаге создается k мыслей, после они оцениваются и выбирается первые n лучших мыслей и уже для каждого из них генерируются новые k мыслей и т.д.

2. DFS
    - Параметры: 
        - count_thoughts: количество генерируемых мыслей на каждом шаге
        - max_depth: максимальная глубина дерева
        - threshold: минимальный порог оценки (если ниже — ветка отсекается)
    - На каждом шаге создается k мыслей, после они оценивают и берется мысль с лучшей оценкой и удолитворяющей минимальному порогу. Далее для неё генерируется k мыслей и т.д.
